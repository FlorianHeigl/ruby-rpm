# build SRPM and RPM

It's Simple.

1. Create your feature branch(e.g ruby-2.2.4)
2. Edit `ruby.spec`
3. Push to the branch.
4. Create a Pull request.
5. When the Pull request is merged, CircleCI will release ruby rpms to https://github.com/feedforce/ruby-rpm/releases .
